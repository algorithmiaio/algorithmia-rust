fn exec<'a, D, E>(&'c self, input_data: &E) -> AlgorithmResult<D> where D: Decodable, E: Encodable

Execute an algorithm with type-safety input_data must be JSON-encodable use #[derive(RustcEncodable)] for complex input

You must explicitly specify the output type T exec will attempt to decode the response into AlgorithmOutput

If decoding fails, it will attempt to decode into ApiError and if that fails, it will error with DecoderErrorWithContext

Examples

let algo_service = Service::new("111112222233333444445555566");
let mut factor = algo_service.algorithm("kenny", "Factor", Version::Latest);
let input = "19635".to_string();
match factor.exec(&input) {
    Ok(out) => {
        let myVal: AlgorithmOutput<Vec<i64>> = out;
        println!("{:?}", myVal.result);
    },
    Err(AlgorithmiaError::ApiError(error)) => {
        println!("API Error: {:?}", error)
    },
    Err(e) => println!("ERROR: {:?}", e),
};
